SQL中case when函数和if函数的区别

在SQL语句中，经常会使用case when函数或者if函数作为条件判断，需要注意的是：

  1. if函数只能用作单个条件判断，case when函数可以用作多个条件判断。
  2. 在使用case when函数的时候一定要加上else，否则不满足case when的其它数据都将被null值所替代。
